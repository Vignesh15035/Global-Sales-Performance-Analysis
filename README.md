# Global-Sales-Performance-Analysis
**Project Overview**

This project presents a Global Sales Performance Dashboard built using MySQL and Power BI to analyze revenue, profitability, regional performance, product contribution, and sales trends.

The dashboard enables stakeholders to monitor business KPIs and make data-driven decisions.

**🎯 Business Objective**

To analyze global sales data and answer key business questions:

What is the total revenue and profit?
Which regions generate the highest sales?
Which products are most profitable?
How do sales trend monthly?
What is the contribution of online vs offline sales?

**🛠 Tools & Technologies Used**

Python (Pandas) – Data Cleaning
MySQL – Data Modeling & ETL
Power BI – Dashboard Development
DAX – KPI Measures
